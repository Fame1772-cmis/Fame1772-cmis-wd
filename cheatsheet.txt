1. ls= <directory name>: List contents of a directory
2. Cd: Go home
3. cd <directory name>: change directory
4. cd ..: Go up
5. cat <file name>: view contents of a file
6. gedit <file name> &: creates a new file or opens up a file that already exists
7. git clone --config="http.proxy=http://uname:pword@192.168.2.1:8080" http://url-to-your-repo: cloning the file to the computer
8. git status: shows what you did/modified
8. git add <file name>: to start tracking new files and also to stage changes to already tracked files
9. git commit -m "Commit Comment": records changes to the repository
10.git push: push your new branches and data to a remote repository
